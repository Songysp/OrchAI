# AI Dev Team Platform

Reusable AI software team orchestration platform with:

- one representative AI for the user,
- internal multi-agent collaboration,
- Slack or Discord transport adapters,
- multi-project configuration,
- storage abstractions ready for future database backends,
- FastAPI endpoints ready for future dashboard use.

See [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) for the platform design.
